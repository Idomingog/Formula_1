# Formula_1
Proyecto con caracter educativo para aprender a realizar webscraping.

El objetivo era obtener todos los resultados historicos de las carreras puntuables de la Fórmula 1, desde sus inicios en 1950 hasta hoy en día.
Se ha desarrollado en Python, y su ejecución obtiene toda la información básica de cada participante en cada carrera, incluyendo todas las carrreras hasta la fecha de ejecución.

Se ha detallado el proyecto en el documento "Contexto Wiki.pdf"

Se ha detallado el código en el documento "Formula_1.pynb"

El fichero de datos obtenido de realizar el webscraping es el "Formula_1_historical_results.csv".
